My Web Apps

Explore the six innovative web applications I have developed, each with its own unique functionality and tech stack.

ArtConnect

ArtConnect is a web app that connects artists and art enthusiasts to showcase artwork, join galleries, and bid on pieces.

Technologies Used:

  • Next.js (Frontend)
  • GraphQL (Backend API)
  • Firebase Firestore (Database)
  • Stripe API (Payments)
  • WebSockets (Real-Time Communication)
  • AWS S3 (Image Storage)

FitTrack Pro

FitTrack Pro is a fitness tracking web app that lets users log their workouts, track progress, and get personalized fitness plans based on their goals.

Technologies Used:

  • Vue.js (Frontend)
  • Python Flask (Backend)
  • PostgreSQL (Database)
  • Fitbit API (Device Integration)
  • Docker (Containerization)
  • Google Cloud (Hosting)

MealMates

MealMates is a meal-planning and recipe-sharing web app that helps users discover new recipes, create shopping lists, and share meal plans.

Technologies Used:

  • Angular (Frontend)
  • Laravel (Backend)
  • MySQL (Database)
  • Spoonacular API (Recipe Data)
  • Firebase (Real-time Data Sync)
  • Heroku (Deployment)

BookNest

BookNest is a social platform for book lovers, offering reading lists, virtual book clubs, and AI-powered book recommendations.

Technologies Used:

  • Svelte (Frontend)
  • Ruby on Rails (Backend)
  • SQLite (Database)
  • OpenAI API (AI Book Recommendations)
  • Elasticsearch (Search Engine)
  • DigitalOcean (Hosting)

Greenify

Greenify is a sustainability-focused web app that helps users track their environmental impact and adopt sustainable habits.

Technologies Used:

  • React Native (Mobile & Web)
  • Firebase (Backend & Authentication)
  • GraphQL (API)
  • MongoDB Atlas (Database)
  • Google Maps API (Eco-Friendly Location Finder)
  • Netlify (Hosting)

EventMaster

EventMaster is a comprehensive event management web app designed for both small and large-scale events. It allows users to create, plan, and track events, manage RSVPs, sell tickets, and provide real-time updates to attendees.

Technologies Used:

  • React.js (Frontend)
  • Node.js (Backend)
  • MongoDB (Database)
  • Express.js (API)
  • AWS (Cloud Hosting)
  • Stripe API (Payment Integration)
